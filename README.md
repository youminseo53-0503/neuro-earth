# 🌍 NEURO·EARTH — 살아있는 인공뇌

지구본 위에 신경 가소성 망(網)을 그리고, **실제 지구 데이터**(실시간 항공 유동인구·지진·위성)로 자극을 주면 헤브 학습으로 스스로 재배선되는 **살아있는 인공뇌**. 회전하는 3D 지구 위에서 노드가 태어나고 죽고, 시냅스가 굵어지고 끊긴다.

> 작년 신경가소성 시뮬레이터(NEURO-SIM)의 연장선 — 같은 헤브 가소성 모델을 **지구 스케일**로 키운 미디어아트 + 인터랙티브 보고서.

화면 오른쪽(데스크탑) / 바텀시트(모바일)에 **개발 과정 전체가 카톡식 로그**로 쌓인다 — 이 웹앱 자체가 보고서다.

## 시나리오

리모컨/하단 바로 전환하거나, **자동순환(전시 모드)**으로 두면 스스로 흘러간다.

- **실시간** — 지금 이 순간의 진짜 항공망으로 살아 학습하는 뇌 (체성·자율 이중 신경계)
- **창세** — 빈 지구에서 거점이 번지고 문명사(기차→자동차→비행기)가 L자로 폭발 성장 (발달·자기조직화)
- **팬데믹** — 우한에서 시작해 항공로를 타고 번지는 SIR 전염 파동 → 대봉쇄 → 회복 → 오늘 (피질 확산성 탈분극)
- **회복** — 손상 후 우회로 재배선 (준비 중)

## 스택

Next.js 16 (App Router) · React 19 · React Three Fiber + drei · three.js · zustand · Tailwind CSS · satellite.js

## 실행

```bash
npm install
npm run dev   # http://localhost:8000
```

> 실시간 항공 데이터를 쓰려면 `.env.local`에 OpenSky 자격증명이 필요하다(없어도 시뮬레이션으로 동작).
